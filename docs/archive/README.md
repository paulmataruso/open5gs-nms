# Documentation Archive

This directory contains development-time documentation and scripts that were used during the creation of Open5GS NMS v1.0 but are not part of the final release documentation.

## Contents

### Development Documentation
- `PROJECT_DOCUMENTATION.md` - Original comprehensive project documentation
- `TECHNICAL_DOCUMENTATION.md` - Detailed technical architecture documentation  
- `DOCUMENTATION.md` - Early documentation attempts
- `TOPOLOGY_PROGRESS.md` - Network topology implementation progress notes
- `AUTO_CONFIG_LEARNINGS.md` - Key learnings from auto-configuration feature implementation

### Feature-Specific Documentation
- `SUBSCRIBER_SCHEMA_FIX.md` - Subscriber schema validation fix notes
- `CLICKABLE_IMSI_NAVIGATION.md` - IMSI navigation feature notes
- `RAN_NETWORK_PAGE.md` - RAN network page implementation notes

### Tooltip System Documentation
- `TOOLTIP_SYSTEM_README.md` - Tooltip system overview
- `TOOLTIP_INSTRUCTIONS.md` - Tooltip implementation instructions
- `TOOLTIP_IMPLEMENTATION_COMPLETE.md` - Tooltip completion notes
- `TOOLTIP_COVERAGE_COMPLETE.md` - Final tooltip coverage documentation
- `QUICK_START_TOOLTIPS.md` - Quick start guide for tooltips

### Build Scripts
- `build-text-editor.sh` - Monaco editor build script
- `add_remaining_tooltips.sh` - Automated tooltip addition script

### Configuration
- `.env` - Development environment configuration (not for production use)

## Note

Most of the content from these files has been consolidated into the official documentation:
- `README.md` - Main project overview
- `INSTALL.md` - Installation guide
- `ARCHITECTURE.md` - System architecture
- `docs/` - Comprehensive documentation directory

These archived files are preserved for historical reference and development context but should not be used as the primary documentation source.
